Application name: MyCart
Submitted by: Nishant Sahoo

Technologies used: NodeJs, ExpressJs, Webpack, Babel, ReactJs, MySQL, sequelize

Installation:

1. Run the following command in the MySQL command line:
CREATE DATABASE shoppingcart
(username: root, password: root)

2. Then, run the file
seedProductsTable.js in the directory API_Node_Folder

3. Then run the command:
npm install
in both the directories

4. Then,
Execute the following command inside the API directory
npm start

5. Execute the following command inside the Front_end_REACT directory
npm start

6. Open the browser and go to http://localhost:8000/

Features: This app can only view the products table
